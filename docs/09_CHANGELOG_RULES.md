# Changelog Rules

## Kapan wajib update dokumen
- Tambah/ubah tabel atau kolom -> update docs/01_DATABASE_SCHEMA_V1.md
- Tambah/ubah route -> update docs/03_ROUTE_MAP.md
- Tambah/ubah API contract -> update docs/04_API_SPEC.md

## Format commit message
- feat: ...
- fix: ...
- chore: ...
- docs: ...
